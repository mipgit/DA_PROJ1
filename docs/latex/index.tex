This project implements a path-\/planning tool designed to assist with urban navigation. Similar to GPS-\/based systems like Google Maps, the tool provides functionality for route planning but with an emphasis on environmentally sustainable and resilient mobility alternatives. The tool offers several key features, including calculating the fastest route, second-\/fastest route, restricted route planning, and environmentally-\/friendly routes that combine both driving and walking.\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Project Motivation}{Project Motivation}}\label{index_autotoc_md3}
The motivation behind this project is to create an urban navigation tool that not only calculates the fastest routes but also optimizes routes based on sustainability by combining driving and walking, while also considering real-\/world constraints. These constraints may include restricted areas, unavailability of routes, and the existence of parking lots. The goal is to provide a versatile solution for diverse mobility needs, helping users plan routes that are faster and more eco-\/friendly.

Key features include\+:
\begin{DoxyItemize}
\item {\bfseries{Independent \doxylink{classRoute}{Route} Planning}}\+: Find the fastest route and the second-\/fastest independent route.
\item {\bfseries{Restricted \doxylink{classRoute}{Route} Planning}}\+: Plan routes that exclude specific nodes or segments of the urban map.
\item {\bfseries{Eco-\/\+Friendly \doxylink{classRoute}{Route} Planning}}\+: Combine driving and walking routes, with parking options, to create environmentally-\/friendly solutions.
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md4}

\begin{DoxyItemize}
\item {\bfseries{Dijkstra’s Algorithm}}\+: Used to find the shortest path between two nodes in the graph (either driving or walking).
\item {\bfseries{Eco-\/\+Friendly \doxylink{classRoute}{Route}}}\+: Combines driving with walking, allowing users to park and then walk to their destination.
\item {\bfseries{Independent Routes}}\+: Finds the fastest route as well as the second-\/best alternative route.
\item {\bfseries{Restricted Routes}}\+: Excludes specific nodes or segments from the route calculation.
\item {\bfseries{Customizable Parameters}}\+: Allows users to set constraints such as maximum walking distance, restricted nodes, and more.
\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{Technologies Used}{Technologies Used}}\label{index_autotoc_md5}
