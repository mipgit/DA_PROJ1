\doxysubsection{Detailed Description}
Class for handling route calculation with restrictions, extending the \doxylink{classRoute}{Route} class. 

This class calculates a route between a source and destination on a given map with specific routing restrictions. It enables users to avoid undesirable areas, excluding specific nodes from the graph. It enables users to avoid undesirable road segments, excluding specific segments from the graph. It enables users to simultaneously exclude the combination of nodes and segments of the graph. It enables users to include a single specific node (or stop) that the route must include while ensuring that the calculated route remains the fastest possible
