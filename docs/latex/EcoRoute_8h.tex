\doxysubsection{Detailed Description}
Contains the definition of the \doxylink{classEcoRoute}{Eco\+Route} class and the the \doxylink{structAproxSolution}{Aprox\+Solution} structure.. 

This file defines the \doxylink{classEcoRoute}{Eco\+Route} class, which calculates environmentally-\/friendly routes that combine driving and walking. The class is capable of calculating an optimal route that minimizes both driving and walking time, while also providing alternative routes when the main route is not feasible, stored in the \doxylink{structAproxSolution}{Aprox\+Solution} structure.
