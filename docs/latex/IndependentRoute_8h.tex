\doxysubsection{Detailed Description}
Contains the definition of the \doxylink{classIndependentRoute}{Independent\+Route} class. 

This file defines the \doxylink{classIndependentRoute}{Independent\+Route} class, which calculates two independent routes between a source and destination. The two routes are independent and share no intermediate nodes or segments except for the source and destination.
