\doxysubsection{Detailed Description}
Class for handling independent route calculation, extending the \doxylink{classRoute}{Route} class. 

This class calculates the best route and an alternative route between a source and destination on a given map. The two routes share no intermediate nodes or segments, except for the source and destination, and that the alternative route is equal to or greater in travel time than the primary route. This provides a robust “\+Plan-\/\+B” option for navigation.
