\doxysubsection{Detailed Description}
Contains the definition of the \doxylink{classRestrictedRoute}{Restricted\+Route} class. 

This file defines the \doxylink{classRestrictedRoute}{Restricted\+Route} class, which calculates a route between a source and destination on a given map. The class allows users to impose specific routing restrictions, such as avoiding certain nodes or road segments and ensuring the inclusion of a specific node. It provides flexibility to control the route based on user-\/defined constraints.
