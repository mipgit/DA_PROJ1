std::string getFullPath ( const std::string &  relativePath)
inline

Resolves the full path of a file based on its relative path.

This function takes a relative file path and combines it with the parent directory of the current working directory to compute the full absolute file path.

Parameters
relativePathThe relative file path to resolve.
Returns
A string containing the full absolute file path.
Exceptions
