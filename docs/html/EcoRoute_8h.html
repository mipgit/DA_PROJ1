Detailed Description

Contains the definition of the EcoRoute class and the the AproxSolution structure..

This file defines the EcoRoute class, which calculates environmentally-friendly routes that combine driving and walking. The class is capable of calculating an optimal route that minimizes both driving and walking time, while also providing alternative routes when the main route is not feasible, stored in the AproxSolution structure.
